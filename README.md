### ğŸ—’ï¸Descripton Of Application ###
The Weather Ambiance is set in a Cute Low Poly 3D Environment where you can escape the hustle and bustle of everyday life and relax and listen to nature sounds. This Project aims to emulate the many YouTube ambiance 3D scenes that have beautiful atmospheric sounds and pleasing visuals. In this ambiant scene there is a day/night cycle which affects whether the birds or crickets play. The time of day also effects the text on the top of the screen which will change to read "Day" or "Night". The camera shakes ever so slightly to simulate the wind blowing against it. There is a rain animation which is affected by the heaviness of the rain set by the user.
